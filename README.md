# AI-Powered-Smart-Blind-Stick-using-ESP32-Ultrasonic-Sensor-YOLO-Blynk
An AI-powered smart blind stick using ESP32 and Raspberry Pi for enhanced mobility assistance. Ultrasonic sensors provide distance-based buzzer alerts, YOLO object detection enables real-time obstacle recognition with voice feedback, and Blynk integration allows emergency alerts for improved user safety.
